# TS-NODE-TOOLING

#### Create config files

 ```bash
    mkdir -p src .vscode &&  \
    touch -a .vscode/settings.json \
    .eslintrc.js .gitignore prettier.config.js \
    babel.config.js tsconfig.json vite.config.js
 ```
